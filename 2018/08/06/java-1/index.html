Java 包

定义和作用

包是java有效管理类的一个机制，不同java源程序可能出现名字相同的累，可以使用包名进行区分

关键字

注意

  1. 如果一个类有包名，则不能在任意位置存放，否则虚拟无法加载，
  2. 存储文件的目录结构要有包相应的结构
  3. 可以将源文件直接放在对应目录下编译，则字节码默认保存在相应目录

运行有包名的主类

比如tom.jiafei
则必须到tom/jiafei 上一级目录执行

abstract 类，abstract 方法

用关键字abstract修饰的类称为abstract类（抽象类）
用abstract修饰的方法称为abstract方法，abstract修饰的方法必须在子类中重写
