基本使用

马克编辑器 是一个全功能的 markdown 编辑器，它包含了以下特性：

  1. 创建，编辑和预览 markdown 文档。
  2. 将你的文档同步到 Dropbox。
  3. 将你的文档通过 Email 发送给其他人。

这个编辑器使用 markdown 语法，让你能够创造格式漂亮的文档。这个语法非常简单，和你使用纯为本几乎没什么区别。
